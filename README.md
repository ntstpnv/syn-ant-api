ТЗ<hr>

Стек: 

- python 
- langchain
- langgraph 
- fastapi

Задание: 

- Пользователь посылает запрос на получение синонимов и антонимов к слову
- Сервер отправляет запрос в ИИ на получение списка 10 синонимов или антонимов к слову пользователя
- Формат запроса: слово, тип (синоним или антоним)
- Полученный ответ отдается пользователю

<hr>
для запуска будет необходимо установить:

- [менеджер зависимостей uv](https://docs.astral.sh/uv/getting-started/installation/)
- [ollama (gemma3:4b)](https://ollama.com/)

uv установит остальные зависимости:
```
uv sync
```

команда запуска приложения:
```
uv run uvicorn app.main:app --reload
```
